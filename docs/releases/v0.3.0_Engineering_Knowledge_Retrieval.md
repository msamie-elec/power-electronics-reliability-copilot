# Release v0.3.0 — Engineering Knowledge Retrieval

## Status

🚧 In Progress

---

# Objective

This release transforms the application from a simple document management system into an AI-powered engineering knowledge retrieval platform.

Rather than only storing uploaded engineering documents, the system will:

- parse engineering documents
- extract readable text
- split documents into semantic chunks
- generate vector embeddings
- build a searchable vector index
- retrieve the most relevant engineering evidence for a user's question

This release forms the Retrieval-Augmented Generation (RAG) foundation for future AI reasoning.

---

# Why this release?

Large Language Models cannot reliably answer engineering questions from documents they have never seen.

This release introduces Retrieval-Augmented Generation (RAG), allowing the AI to search project-specific engineering documentation before generating responses.

The output of this release will become the primary knowledge source for GraphRAG (v0.4.0) and Agentic AI workflows (v0.5.0).
